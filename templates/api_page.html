API Documentation


API Call:


f'/api/accidents
' f'/api/accidents/min_severity
' f'/api/states/state
' f'/api/zipcode/zipcode

Response Class:


City (string): Name of city
Description (string): Accident description
Latitude (int): Coordinate
Longitude (int): Coordinate
Severity (int): Level from 1 to 4
State (string): State abbreviation
Street (string): Street address
Temperature(F) (string): Temperature in Farenheit
Time (string): Includes date and time of accident
Timezone (string): US timezones only
Visibility(mi) (string): ****
Weather_Condition (string): ****
Zipcode (string): Address zipcode
_id (string): Unique accident ID
